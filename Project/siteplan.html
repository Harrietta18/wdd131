LagosHub Connect Site Plan

Erhurhe Harrietta

Site Name

LagosHub Connect

This name was chosen because it highlights the geographical focus (Lagos) and the site's primary function of connecting professionals to tech resources.

Site Purpose

To provide a high-quality, interactive directory of tech hubs in Lagos. It aims to help tech workers find spaces that match their specific needs for power, internet, and community.

Scenarios

  • "Which tech hubs in Yaba offer 24/7 access for late-night developers?"
  • "Where can I find a co-working space with meeting rooms for a client presentation?"

Color Schema

Primary: #008751 (Nigerian Green)
Secondary: #FFBE0B (Amber)
Accent: #333333 (Dark Grey)

The green represents our national identity, while the amber provides high-contrast call-to-action buttons.

Typography

Headings: 'Montserrat', sans-serif - A clean, bold font for a modern tech feel.

Body: 'Open Sans', sans-serif - Optimized for readability across all device sizes.
