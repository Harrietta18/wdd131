LagosHub Connect: Site Plan

Site Name

LagosHub Connect

I chose this name because it directly identifies the location and the core service (connecting people to hubs).

Site Purpose

To provide a searchable, dynamic directory of innovation hubs in Lagos to help tech professionals find reliable workspaces.

Scenarios

  • Where can I find a hub in Lekki with 24/7 power?
  • Which hubs offer private meeting rooms for team sessions?

Color Schema

Primary: #008751 (Headings) | Secondary: #F4F4F4 (Background)

Typography
